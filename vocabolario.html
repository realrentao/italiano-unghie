⚡ 语法小贴士：定冠词 Il / La / L' / Lo

意语名词分"男"和"女"，前面都要加定冠词：

Il + 男名词（辅音开头）→ il centro estetico
La + 女名词（辅音开头）→ la cliente
L' + 元音开头 → l'estetista
Lo + s+辅音/ps/gn/z → lo smalto

🖐️ 甲质与状况描述（美甲师必背！）

Le unghie sono deboli / fragili指甲很脆/很弱
Le unghie sono sottili指甲很薄
Le unghie sono morbide指甲很软
Le unghie sono dure指甲很硬
Le unghie si rompono指甲容易断
Le unghie si sfaldano指甲分层/起皮
Le unghie sono rovinate指甲受损
Le unghie sono mangiate指甲被咬过
Le cuticole secche死皮干燥
Le cuticole infiammate死皮发炎/红肿
La macchia bianca白点
L'onicomicosi灰指甲
⚡ 句型公式：Le unghie sono + 形容词
Le unghie sono deboli → 指甲是脆的
Le unghie sono sottili → 指甲是薄的
Le unghie sono rovinate → 指甲是受损的

记住三个核心词：脆的 → deboli | 薄的 → sottili | 受损的 → rovinate
